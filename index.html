CSS Snippet Cheat Sheet

Ever have trouble recalling the exact syntax for your favorite CSS code? Give it a permanent home, and add it to this page! Select any snippet below, and it will automatically select all of the code for you to copy.

Overlay Title on Card Border

Use position properties and margins to raise elements higher than their natural starting point and center them. 

.card h2 {
  position: relative;
  margin: auto;
  top: -20px;
  left: 0;
  right: 0;
  width: 95%;
}

Linear Gradient

This will create a background linear gradient from top to bottom.

.card {
  background-image: linear-gradient(
    color-1 0%,
    color-2 100%
    );
}

Box Transition Glow

Use transition and box shadows to glow on hover.

.card {
  transition: all 0.5s ease-in-out;
}

.card h2 {
  transition: all 0.5s ease-in-out;
}

.card:hover, .card:hover h2 {
  box-shadow: inset 0 0 8px color, 0 0 15px color;
}
